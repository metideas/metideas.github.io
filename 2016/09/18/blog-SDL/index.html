设置基于 Github 网站的子域名和wiki系统搭建

Sep 18, 2016

设置域名下的子域名. 有这个想法很长时间了, 但是一直没有去折腾. 其实之前搭建过Github page的项目页面, 直接导入到mydomain/wiki 下面了, 一直也没有成功的搭建子域名. 这回终于要把这个事完成了.

由于我的域名是从Hostgator 购买的, 其他那些直接在 Godday 上设置子域名的教程都是不适用的, 而hostgator 好像只提供给cpanal 各种管理域名的权限…也就是说你要在他家买host plan…囧了
如图所示, 光买domain 根本没有domain管理选项.
好了, 那就只能仰仗良心公司Github 了. 首先创建了一个新的 repository, 创建 Github page, 并且自己设置好域名, 例如我设置的wiki.metideas.com

需要学习一个:
Github
Git
javascript
npm
react
等等 不一而足

18-09-16
暂时终止搭建.. 原因是wiki主题出现了问题…不知道什么时候继续折腾这个东西…蛋疼了

暂时还是用evernote 记录有价值的东西吧, 可能并没有那么紧迫…囧
