## Syntax
- `<q>`: quote, 用于显示引用, 也方便 CSS 中修改 quota 的模式
- `<blockquote>`:块引用
> b 区别 blockquote 的元素是块(block), 而q 的元素是内敛( Inline)
- `<br>`:换行
- `<em>` 斜体
- HTML 中元素分成两类, void elements(`<br>` `<img>`) & normal elements(`<a>` `<p>`)等
- `<li></li>`: 列表元素, 并用`<ol>``<ul>`元素封装
